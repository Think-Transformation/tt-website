# Think Transformation Inc Landing Page

A landing page for Think Transformation Inc, the parent company managing Experiencing Yah and other ventures.

## Features

- **Hero** with "Transform Your Thinking" tagline
- **Development section** with information about Experiencing Yah
- **Contact form** integrated with Formspree
- **Brand colors**: Yellow, Purple, White (from TT logo)

## Setup

1. Push to GitHub and enable GitHub Pages
2. Add custom domain `thinktransformationinc.com` in repository settings
3. Configure Formspree form endpoint if using a separate form from Experiencing Yah
